Role Management

Granular access control,
effortlessly managed.

Create unlimited roles with fine-grained privileges. Check permissions in controllers, middleware, Blade templates, or anywhere in your code with intuitive helpers.

$user->hasRole('admin')
$user->hasPrivilege('reports.run')
$user->can('billing.view')
UserController.php
// Check if user has admin role if ($user->hasRole('admin')) { // Grant access } // Check specific privilege via Gate if ($user->can('reports.run')) { // User can run reports } // Require all listed privileges $hasAccess = $user->hasPrivileges([ 'reports.run', 'billing.view' ]);
Powerful Middleware

Protect routes
with precision.

Eight purpose-built middleware aliases let you enforce role and privilege checks directly on your routes. Combine with Laravel's auth middleware for complete protection.

role:admin
roles:editor,admin
privilege:reports.run
abilities:admin,edit
routes/api.php
use Illuminate\Support\Facades\Route; // Require admin role Route::middleware([ 'auth:sanctum', 'role:admin' ])->get('dashboard', DashboardController::class); // Allow editor OR admin Route::middleware([ 'auth:sanctum', 'roles:editor,admin' ])->post('articles', ...); // Require specific privilege Route::middleware('privilege:reports.run') ->get('reports', ...);

Sanctum Tokens

Security that
never sleeps.

Sanctum tokens automatically mirror role and privilege slugs. Suspension workflows revoke tokens instantly. Emergency token rotation is one command away.

Token abilities include role and privilege slugs
Suspension instantly revokes all active tokens
Per-user slug caching for optimal performance
AuthController.php
// Suspend a user (revokes all tokens) $user->suspend('Pending review'); // Check suspension status if ($user->isSuspended()) { $reason = $user->getSuspensionReason(); } // Unsuspend and restore access $user->unsuspend(); // Emergency token rotation // $ php artisan tyro:auth-logout-all-users --force
Quick Start

Up and running
in seconds.

Install Tyro, run one command, and you have a complete auth system.

1

Install the package

Pull in the core auth engine via Composer.

composer require hasinhayder/tyro
2

Run the installer

Sets up Sanctum, runs migrations, seeds roles/privileges, and prepares your User model.

php artisan tyro:sys-install
3

Start using Tyro

Protect routes, check permissions, render Blade directives. You're ready to ship.

Route::middleware(['auth:sanctum', 'role:admin']) ->group(function () { // Protected routes });
if ($user->can('reports.run')) { // User has the privilege } if ($user->hasRole('admin')) { // User is an admin }
@hasRole('admin') <div class="admin-panel"> Welcome, Admin! </div> @endhasRole @userCan('edit-posts') <button>Edit Post</button> @enduserCan
